基本类型

V语言中共有20个基本类型，如下:

布尔类型(Boolean)

  • bool

字符串(String)

  • string

整数(Integer)

  • 有符号整数: i8、i16、i32 、i64
  • 无符号整数: u8、u16、u32、u64
  • 别名: byte(u8)、 int(i32)、 rune(i32，表示Unicode代码点)

浮点数(Float)

  • f32、f64

*请注意，与C和Go不同， V语言中int始终是32位整数。
