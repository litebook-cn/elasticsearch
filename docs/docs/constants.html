常量(Constants)

const (
    PI = 3.14
    WORLD = '世界'
)

fn main() {
    println(PI)
    println(WORLD)
}

使用 const关键字来申明常量，常量定义的位置只能在模块级别(函数外)。

常量名称必须大写， 这有助于将它们与变量区分开来。

常量的值一经定义，永远不能修改。

V语言中的常量比大多数语言更灵活，你可以指定更复杂的值:

struct Color {
        r int
        g int
        b int
}

fn (c Color) str() string { return '{$c.r, $c.g, $c.b}' }

fn rgb(r, g, b int) Color { return Color{r: r, g: g, b: b} }

const (
        NUMBERS = [1, 2, 3]

        RED  = Color{r: 255, g: 0, b: 0}
        BLUE = rgb(0, 0, 255)
)

fn main() {
        println(NUMBERS)
        println(RED)
        println(BLUE)
}

V语言中没有全局变量，常量就更加有用了。
